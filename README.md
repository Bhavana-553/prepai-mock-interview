# PrepAI — AI Mock Interview Platform

A full-stack AI-powered mock interview platform built for job seekers to practice interviews and get instant AI feedback.

## 🚀 Tech Stack
- **Frontend:** React, React Router, Recharts, Axios
- **Backend:** Node.js, Express, JWT Authentication
- **Database:** PostgreSQL
- **AI:** Groq (LLaMA 3.3 70B)

## ✨ Features
- AI-generated interview questions by topic
- Speech-to-text answer input
- AI-based scoring and feedback
- Performance analytics with charts
- JWT-secured authentication

## ⚙️ Setup
1. Clone the repo
2. Create `.env` in `/server` with your credentials
3. Run `npm install` in `/server` and `/frontend`
4. Run `node server.js` in `/server`
5. Run `npm start` in `/frontend`
